Chimie API

Docs

Base URL: "https://chimie-api.herokuapp.com"

Get list of all elements

To get the entire list of elements along with some of their data, follow the base route with "/api/elements" added to URL

The json response will look similar to this:

{"elements": [
{ "atomic_number": 1,
"mass": 1.008,
"name": "Hydrogen",
"symbol": "H" },
{ "atomic_number": 2,
"mass": 4.0026,
"name": "Helium",
"symbol": "He" },
{ "atomic_number": 3,
"mass": 6.94,
"name": "Lithium",
"symbol": "Li" }
]}

Get details of one element

To get a single element with all info, follow the base route with "/api/elements/[id]" added to the URL where id is the atomic number.

Axios request in javascript to get data for helium would look like this:

heliumData = await axios.get('https://chimie-api.herokuapp.com/api/2')

And the json response for a single element will look like this:

{"element": {
"atomic_number": 1,
"mass": 1.008,
"name": "Hydrogen",
"symbol": "H",
"group": "noble gas",
"boiling_point": 4.22,
"melting_point": .95,
"year_discovered": 1868
}}
